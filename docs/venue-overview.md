# Berlin Beach Volleyball Venue Dataset

Research date: 2026-04-24

This overview is generated from `data/venues/paid.json` and `data/venues/free.json`. Duplicate listings from Urban Sports Club, TIP Berlin, BeachUp, Berlin-Beachvolleyball.de, Parkinspector, and official venue pages are merged into canonical venue IDs where they describe the same real venue. English venue summaries live in `description`; structured facts live in `facts`; German Parkinspector source text lives in `data/venues/de/venues.json`.

## Image Rights Notes

Most official venue and publication photos are copyright protected and should be treated as permission-required even when a source/copyright line is known. The current immediately usable images are the Wikimedia Commons records marked `usable` in the JSON data. Parkinspector images are tracked as `requires_permission` because no reusable license was found.

## Parkinspector Import Notes

Imported only entries with `cityname = Berlin` from the Parkinspector Beachen API. Excluded Glienicke because it is outside Berlin and excluded TU Berlin Campus because Parkinspector marks it closed. Existing records for Monbijoupark, Schendelpark, Schreinerstraße, Volkspark Friedrichshain, Volkspark Jungfernheide, Newtonstraße, and Landschaftspark Johannisthal were merged instead of duplicated.
